# خلاصه تغییرات
- پیاده‌سازی صفحات احراز هویت (ورود/ثبت‌نام) با کپی فارسی و اعتبارسنجی کامل.
- افزودن bootstrap نشست، ذخیره‌سازی سشن یکپارچه و گاردهای نقش‌محور برای پنل کاربری و ادمین.
- ایجاد رجیستری ناوبری، فلگ‌های ویژگی و منوی پایین برای موبایل به‌همراه «بیشتر».

# نحوه تست
- pnpm lint
- pnpm typecheck
- pnpm build

  - ثبت‌نام: /register
  - ورود: /login
  - دسترسی پنل کاربری: /trader
  - دسترسی پنل ادمین: /admin
